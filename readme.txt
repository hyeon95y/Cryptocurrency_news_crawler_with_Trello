코인 리스트 업데이트 하는 방법

0. 수정할 내용이 있으면 Coinlist.gsheet을 수정후 저장
1. Coinlist.gsheet에 업데이트 된 내용이 있으면 Coinlist.gsheet의 데이터 필드 전체를 복사
2. Excel을 켜고 붙여넣기
3. 저장->파일 형식 csv선택
4. Coinlist.csv로 같은 폴더에 저장(모두 예)
5. Coinlist.csv를 우클릭 -> 연결프로그램 -> 메모장으로 열기
6. 다른 이름으로 저장 -> 파일형식 : 모든 파일 -> 인코딩 -> UTF-8로 변경 -> Coinlist.csv 덮어쓰기로 저장
